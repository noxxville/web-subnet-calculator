# Subnet Calculator

A modern IPv4 subnet calculator with CIDR analysis and tree-based subnet splitting.

## Features
- IPv4 CIDR analysis
- Tree-based subnet splitting (+1)
- Usable IP & host calculation
- Copy-to-clipboard
- Dark / Light theme
- Fast, local, no tracking

## Tech
- Vanilla React (UMD)
- Tailwind CSS
- No backend
- No cookies
